RNN-Based Turbofan Degradation Forecasting

This project developed a Recurrent Neural Network (RNN) model to predict the Remaining Useful Life (RUL) of turbofan engines. Accurate RUL prediction is crucial for predictive maintenance, enabling timely interventions and preventing failures. The model was trained on NASA's C-MAPSS dataset, which provides simulated sensor data of engine degradation. RNNs were chosen for their ability to capture temporal dependencies in the data. The project involved data preprocessing, RNN architecture selection, model training, and validation. The trained RNN effectively learned complex degradation patterns, outperforming baseline models in RUL prediction accuracy.

  • Tech Used: Python, Machine Learning, Recurrant Neural Network,NASA's CMAPS Data Set

RE-Based RUL Estimation for Turbofans

Residual ensemble models leverage multiple deep learning networks with skip connections to analyze engine data, capturing complex degradation patterns. This approach provides more accurate and reliable predictions of an engine's remaining useful life. methods like gradient boosting and random forest regression are used.

  • Tech Used: Python, Machine Learning,Gradient boost, Random Forest Regression, NASA's CMAPS Data Set
